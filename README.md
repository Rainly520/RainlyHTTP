# RainlyHTTP 支持断点传输的下载上传程序
一款轻量跨平台的文件上传下载服务工具，无需复杂配置，直接运行可执行文件即可快速部署，为你提供安全、高效的文件传输解决方案。

✨ 核心特性

> 🚀 跨平台运行
完美支持 Windows 和 Linux 系统，无需额外依赖，下载对应平台可执行文件，双击 / 命令行启动即可使用。

> 📤 断点续传
大文件传输不怕中断！支持上传 / 下载断点续传，网络波动后无需重新传输，节省时间与带宽。

> 🔒 密码加密上传
支持上传密码验证，只有输入正确密码才能上传文件，有效防止未授权文件上传，保障服务安全。

> ⚙️ 高度自定义
自定义监听 IP 和 端口，灵活适配不同网络环境
限制上传文件大小，避免超大文件占用服务器资源

> 🛡️ 安全防护
自动过滤非法请求参数，有效抵御常见的恶意请求，降低服务被攻击的风险。

# 📖 使用说明
文件下载：浏览器访问 http://[监听IP]:[端口]/，即可查看并下载服务端文件
文件上传：访问上传页面，输入预设密码后，选择文件即可上传（支持断点续传）

下载示例：http://172.16.0.1/文件名（例：http://172.16.0.1/1.zip）


上传示例：POST http://172.16.0.1/upload（form-data 格式，字段名 file）


头部Header携带X-Upload-Password参数 或者 从 URL 参数获取（如 ?password=xxx）
